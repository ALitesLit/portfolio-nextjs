# **Инструкция к сайту-портфолио**
Основные команды npm:
```bash
npm run dev # Запуск для разработки
npm run build # Сборка проекта
npm run start # Запуск для продакшена
npm run test # Запуск тестирования Jest
```

Основные команды Docker:
```bash
sudo docker build -t portfolio-client . # Сборка в контейнер
sudo docker compose up -d # Запуск через docker compose
```

# **Краткая информация о стеке**
 Для front-end части сайта-портфолио в этой версии используется:
- Next.js
- React
- Redux
- TypeScript
- Tailwind
- Jest (для автоматического тестирования)
- Sass
- Axios
- Docker
